PANOOJA LANDING PAGE

How to open:
1. Double-click index.html.
2. The page will open in your browser.

How to publish free:
- Vercel: drag and drop this folder into a new project or upload it to GitHub and import it.
- Netlify: use Netlify Drop and drag this folder.
- Hostinger: upload all files into public_html.

Important before launch:
- Replace the text PANOOJA wordmark with your official logo image if required.
- Connect the waitlist form to Formspree, Brevo, Mailchimp, Supabase, or another real email database. The current demo stores entries only in the visitor's browser.
- Verify the Instagram username in index.html.
- Add final product photographs, pricing, policies, WhatsApp contact, and launch date when available.
